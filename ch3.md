#### if,if-else,if-else if-else 指令
- 條件式內不必加分號
- 指令超過兩行必要用大括號
- else if後面要接條件式
- else 後面不用加條件式
- 頭一個為if
- 中間皆用else if
- 最後用一個else
- else if和else皆不是必要存在
